Simplify a give path

Input: String. Unix path.

Output: String. Unix path.

Example: 

{% if interpreter.slug == "js-node" %}
sumTwo(2, 3) == 5
sumTwo(5, 7) == 12
{% else %}
simplify_path('/a/') == '/a'
simplify_path('/a/b') == '/a/b'
simplify_path('/a//b/c') == '/a/b/c'
simplify_path('/a/b/./ci') == '/a/b/ci'
simplify_path('/a/b/../c') == '/a/c'
{% endif %}
